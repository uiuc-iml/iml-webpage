ROBOPuppet is a method to create inexpensive, table-top sized models of robots to provide teleoperate input control to full sized robots. The ROBOpupept method involves rapid prototyping a minature replica of your target robot using a 3d printer, installing easy to use electronic components and connecting everything together with a motion planning toolkit(Klampt't).
